# 울산공업고등학교 도제반 관리 시스템
## Ulsan Technical High School Apprenticeship Management System

### 📋 프로젝트 개요

울산공업고등학교 일학습병행제(도제반) 학생 관리를 위한 전문 웹 기반 시스템입니다.

**주요 기능:**
- 77명 학생 관리 (3학년 38명, 2학년 39명)
- 40개 기업체 취업 현황 관리
- 자격증 취득 현황 추적
- 상벌점 시스템
- 출결 관리 (학교 + 기업 이중 관리)
- 공지사항 및 가정통신문 관리
- **도제 프로그램 소개**
- **연간 행사 계획 (12개월)**
- **월별 캘린더**
- **교사 소개 (8명)**

---

## 🚀 시작하기

### 실행 방법

1. **파일 열기**
   - `index.html` 파일을 더블클릭하거나
   - 웹 브라우저에서 직접 열기

2. **로컬 서버 사용 (권장)**
   ```bash
   # Python이 설치되어 있다면
   python -m http.server 8000
   
   # 또는 Node.js가 설치되어 있다면
   npx -y http-server -p 8000
   ```
   
   그 후 브라우저에서 `http://localhost:8000` 접속

### 필요 사항

- 최신 웹 브라우저 (Chrome, Edge, Firefox, Safari)
- 인터넷 연결 (Google Fonts 및 Chart.js CDN 사용)

## 🎯 주요 기능

### 1. 도제 소개 (첫 페이지)
- **도제란 무엇인가**: 일학습병행제 설명
- **도제의 특징**: 6가지 주요 특징 (이론+실무, 멘토링, 훈련수당, 자격증, 취업보장, 경력인정)
- **운영 방식**: 주간 일정 및 교육 과정
- **성과**: 취업률 100%, 협약기업 40개, 평균 자격증 2.8개

### 2. 대시보드 (홈)
- 전체 학생 수 및 학년별 통계
- 취업률 차트
- 최근 공지사항
- 학급별 현황 요약

### 3. 학급 현황
- 학급별 학생 카드 그리드
- 학생 클릭 시 상세 프로필 모달
- 사진, 이름, 학급, 취업 기업 표시

### 4. 학생 상세 프로필
**연락처 정보:**
- 본인, 부모님, 담임교사 연락처

**도제 현황:**
- 기업체명, 주소, 업종
- 담당 멘토 정보 (이름, 직급, 연락처)
- 시작일

**취득 자격증:**
- 자격증명 및 취득일
- 선반기능사, 밀링기능사, 금형기능사 등

**상벌점 현황:**
- 상점/벌점 그래프
- 총점 계산
- 상벌점 이력 (날짜, 사유)

**상담 기록:**
- 상담 일자 및 주제
- 상담 내용 기록

**출결 현황:**
- 최근 30일 기록
- 학교 출석 + 기업 출근 이중 관리
- 출석, 지각, 결석, 현장실습 상태

### 5. 취업 정보
- 기업별 배치 현황
- 40개 기업 정보 (이름, 업종, 주소)
- 학생별 취업 현황 테이블
- 멘토 정보

### 6. 자격증
- 자격증 종류별 취득 현황
- 학생별 자격증 목록
- 취득 통계

### 7. 연간 행사 (NEW)
- **12개월 전체 일정**: 3월부터 2월까지
- **행사 카테고리**: 학사, 도제, 안전, 행사, 자격증, 평가, 휴무, 교육
- **주요 행사**:
  - 개학식/종업식
  - 도제 기업 오리엔테이션
  - 산업안전보건교육
  - 자격증 시험 주간
  - 기업 평가회
  - 취업 박람회
  - 도제 성과 발표회

### 8. 월별 캘린더 (NEW)
- **인터랙티브 월 선택**: 12개 월 버튼
- **색상 코딩**: 행사 유형별 색상 구분
- **상세 정보**: 날짜, 제목, 설명
- **현재 월 강조**: 자동으로 현재 월 표시

### 9. 교사 소개 (NEW)
**도제부 구성 (8명)**
- 도제부장 1명
- 담임교사 4명
- 전담관 3명

**교사 프로필 정보:**
- 사진
- 이름 및 직책
- 담당 학급 (담임교사)
- 전공 분야
- 경력
- 담당 업무
- 소개
- 연락처 및 이메일

**도제부장:**
- 김철수 (기계가공, 15년 경력)

**담임교사:**
- 이영희 (3-4반, 금형제작, 12년)
- 박민수 (3-5반, CNC가공, 10년)
- 최지영 (2-4반, 기계설계, 8년)
- 정현우 (2-5반, 용접, 11년)

**전담관:**
- 강수진 (도제 행정, 7년)
- 조민호 (취업 지도, 9년)
- 윤서연 (자격증 관리, 6년)

### 10. 교사 관리
**출석 관리:**
- 일일 출석률
- 지각/결석/현장실습 현황
- 출석 입력 인터페이스

**상벌점 관리:**
- 학생별 상벌점 현황
- 실시간 점수 수정
- 점수 이력 관리

**상담 기록:**
- 상담 내역 추가 및 관리

### 11. 공지사항
- 카테고리별 공지 (긴급, 도제, 일반, 행사)
- 날짜 및 작성자 표시
- PDF 업로드 기능 (UI 준비됨)

### 12. 가정통신문
- 발송 현황 관리
- 확인률 추적
- 학급별 일괄 발송
- PDF 업로드 및 관리

## 📝 사용 팁

1. **도제 소개 보기**: 첫 페이지에서 도제 프로그램 전체 설명 확인
2. **학생 프로필 보기**: 학급 현황 페이지에서 학생 카드 클릭
3. **모달 닫기**: X 버튼 클릭 또는 ESC 키 또는 배경 클릭
4. **페이지 이동**: 상단 네비게이션 바 사용
5. **월별 일정 보기**: 월별 캘린더에서 원하는 월 버튼 클릭
6. **교사 정보 확인**: 교사 소개 페이지에서 상세 프로필 확인
7. **데이터 확인**: 브라우저 콘솔에서 `window.AppData` 확인
